# **App Name**: GlobalTalent Careers Portal

## Core Features:

- Job Search: Allows users to search for jobs by category, country, pay rate, and experience level.
- Job Application: Multi-step application form with real-time validation and document upload.
- Application Tracking: Applicants can track their submission status with a reference number.
- Intelligent Qualification Verification Tool: AI Tool which reads the resume and compares the details with the 'Required qualifications' to help the applicant better judge their qualification before manually completing the application.

## Style Guidelines:

- Primary color: Deep Navy Blue (#0D1B2A) to convey authority and professionalism.
- Accent color: Amber/Gold (#D4AF37) to highlight key elements and create a sense of prestige.
- Background color: Light Gray (#F4F6F9) for a clean and structured look.
- Headings: 'Playfair' serif font to establish a formal and authoritative tone.
- Body: 'Inter' sans-serif font for clean and readable text.
- Use of professional and easily recognizable icons for job categories and countries.
- Structured and clean layout with horizontal gold divider lines between sections to create a clear hierarchy.
